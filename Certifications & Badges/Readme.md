# Cloud Learning Labs – README

## Overview

This repository is a collection of **hands-on cloud labs paired with certification-focused theory**. Rather than acting as a pure knowledge base, it is structured to **introduce key certification topics and reinforce them through practical exercises**.

Each document aligns with real-world cloud concepts while reflecting the type of material covered in entry-level and associate-level cloud certifications.

---

## What This Repository Focuses On

* Guided **labs and simulations** to apply concepts in practice
* **Certification-aligned topics**, especially foundational cloud knowledge
* Supporting **theory explained in context**, not in isolation
* Building intuition for how cloud services are used in real scenarios

---

* **Cloud Computing Essentials** – Core principles behind cloud platforms and service models

## How to Use This Repository

1. Start with **Cloud First Steps** to get familiar with the environment
2. Move through labs by category (compute, networking, storage, etc.)
3. Focus on **doing the labs**, not just reading the theory
4. Use the theory sections as **supporting context for certification topics**
5. Revisit labs when preparing for exams to reinforce understanding

---

## Key Idea

This repository is designed around a simple principle:

> Learn the certification topics by **doing**, not just reading.
