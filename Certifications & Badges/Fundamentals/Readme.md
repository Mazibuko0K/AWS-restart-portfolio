# Core Cloud Concepts – README

## Overview

This section introduces foundational cloud topics through **guided labs**, with just enough theory to support certification understanding.

## What You’ll Do

* Set up and explore basic cloud environments
* Understand core service models through hands-on usage
* Learn essential terminology in context

## Labs Included

* Cloud First Steps
* Cloud Computing Essentials

## Focus

* IaaS, PaaS, SaaS in practice
* Regions, availability zones, and basic architecture
* Navigating a cloud provider environment

## Outcome

Build a working baseline for all other labs and certification topics.
