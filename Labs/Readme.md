# Labs Directory – README

## Overview

This directory contains **hands-on labs organized by domain**, each focused on building practical skills while reinforcing certification-relevant concepts. The emphasis is on **learning by doing**, with theory included only where it supports the task.

---

## Sections

### 🖥️ Compute

Focuses on deploying and managing compute resources in the cloud.

**What you’ll do:**

* Launch and configure virtual machines
* Explore compute options (VMs, containers, serverless)
* Understand scaling and availability basics

**Outcome:**
Ability to provision and manage compute services in real scenarios.

---

### 💾 Databases

Covers database operations through practical labs.

**What you’ll do:**

* Work with managed database services
* Perform basic database operations
* Explore NoSQL concepts

**Outcome:**
Understand how to deploy and interact with cloud-based databases.

---

### 🐧 Linux / Editing Files

Builds foundational command-line and file management skills.

**What you’ll do:**

* Navigate Linux environments
* Edit and manage files via CLI tools
* Understand basic system operations

**Outcome:**
Gain essential skills required for working in most cloud environments.

---

### 🌐 Networking

Focuses on connectivity and communication between resources.

**What you’ll do:**

* Configure networking components
* Work with IP addressing and routing basics
* Connect isolated environments

**Outcome:**
Understand how cloud resources communicate securely and efficiently.

---

### 🔐 Security

Introduces core security practices through applied labs.

**What you’ll do:**

* Configure identity and access controls
* Apply basic security best practices
* Understand shared responsibility

**Outcome:**
Build a practical foundation in securing cloud resources.

---

### 🐍 Python

Provides scripting and automation support for cloud tasks.

**What you’ll do:**

* Write basic Python scripts
* Automate simple workflows
* Interact with cloud services programmatically

**Outcome:**
Use Python to improve efficiency and automate repetitive tasks.

---

## How to Use

* Start with **Linux / Editing Files** if unfamiliar with command-line environments
* Progress through sections based on your focus area or certification goals
* Complete labs actively; avoid skipping steps
* Use theory sections only as support for completing tasks

---

## Key Principle

These labs are designed to **simulate real usage**, not just demonstrate concepts. Focus on completing each task and understanding the outcome.
